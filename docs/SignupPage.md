# Signup Page

## Route
`/signup`

## What's on this page
A registration form for new users to create an account.

## Fields
- **Full Name** — your display name
- **Email** — your email address
- **Password** — pick a secure password
- **Confirm Password** — type it again to make sure

## Features
- Password matching check — shows a warning if passwords don't match
- Submit button stays disabled until all fields are valid
- "Log in" link at the bottom for existing users
- Same theme and design language as the rest of the app
- Fade-in animation on load

## Notes
Just like the login page, this is visual only — no accounts are actually created yet. Ready for backend integration when needed.
