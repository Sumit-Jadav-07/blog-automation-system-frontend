# Login Page

## Route
`/login`

## What's on this page
A clean login form where users enter their email and password to sign in.

## Fields
- **Email** — where you type your email address
- **Password** — where you type your password

## Features
- Form validation (both fields must be filled to submit)
- "Sign up" link at the bottom for new users
- Same warm, earthy design as the rest of the app
- Fade-in animation when the page loads

## Notes
This page is visual only for now — no real authentication happens behind the scenes. It's ready to be connected to a backend auth system later.
