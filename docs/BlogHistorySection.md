# Blog History Section

## Where it appears
Left column of the Dashboard page (below the Blog Form).

## What it shows
A scrollable list of all previously generated blogs. Each item shows:

- **Title** — the blog's title (or the original topic if no title yet)
- **Topic** — what the blog is about
- **Status Badge** — draft, generating, published, or failed
- **Creation Date** — when the blog was created
- **Read Link** — opens the full blog reader page

## Interactions
- **Click a blog** → selects it and shows it in the preview panel on the right
- **Click "Read →"** → navigates to the full blog reader page at `/blog/:id`

## Empty State
When there are no blogs yet, it shows a friendly message: "No blogs yet. Generate your first topic to populate the queue."

## Animations
Each blog item has a stagger animation — they slide in one after another for a nice cascading effect.
